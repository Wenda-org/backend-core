## 📍 Uso para ML e Recomendações

### Dados Estruturados
Todos os destinos possuem:
- ✅ Nome e slug únicos
- ✅ Descrição
- ✅ Província e localização
- ✅ Categoria
- ✅ Coordenadas GPS (latitude/longitude)
- ✅ Rating (0-5)

### Casos de Uso

**1. Recomendações Personalizadas**
```typescript
// Exemplo: Usuário interessado em praias
const praiaDestinations = await prisma.destination.findMany({
  where: { category: { slug: 'praia' } },
  orderBy: { rating: 'desc' }
});
// Retorna: Baía Azul, Ilha do Mussulo, Praias de Benguela, Praias de Cabinda
```

**2. Busca Geográfica**
```typescript
// Destinos próximos a Luanda
const nearbyDestinations = await prisma.destination.findMany({
  where: { province: 'Luanda' }
});
```

**3. Filtro por Avaliação**
```typescript
// Destinos top-rated
const topDestinations = await prisma.destination.findMany({
  where: { rating: { gte: 4.5 } },
  orderBy: { rating: 'desc' }
});
// Retorna: 10 destinos com rating >= 4.5
```
